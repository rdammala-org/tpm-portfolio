# TPM Portfolio — Rajesh Dammala

A clean, professional portfolio website showcasing my experience as a Technical Program Manager specializing in cloud infrastructure, SRE programs, and AI-driven automation.

## 🚀 Live Site

[https://rdammala-org.github.io/tpm-portfolio](https://rdammala-org.github.io/tpm-portfolio)

## 📁 Structure

```
tpm-portfolio/
├── index.html    — Main portfolio page
├── style.css     — Styles (light theme, responsive)
├── script.js     — Navigation & scroll animations
└── README.md     — This file
```

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)
- GitHub Pages (hosting)

## 📋 Sections

- **Hero** — Introduction with program dashboard visual
- **About** — Summary with leadership pillars
- **Expertise** — Areas of expertise grid
- **Experience** — Career timeline (TPM-framed)
- **Impact** — Key program metrics & outcomes
- **Education & Certifications**
- **Contact** — Contact information

## 🏗️ Local Development

Simply open `index.html` in a browser. No build tools required.
